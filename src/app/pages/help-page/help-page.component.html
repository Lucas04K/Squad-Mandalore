Eigener Account

Login

Falls Sie die Seite zum ersten Mal nutzen wollen, benutzen Sie bitte den vorgegebenen Standard Admin Nutzer.

Falls Sie nicht der Admin sind und die Webapplikation zum ersten Mal nutzen wollen, müssen Sie die Login Daten bei Ihrem Admin anfragen.

Nach dem Ersten Login werden Sie gebeten das Passwort zu ändern.
Ist das geschehen, können Sie sich ab sofort mit Ihrem neuen Passwort anmelden.

Passwort vergessen

Falls Sie ihr Passwort vergessen sollten, ist ihr Nutzerkonto unwiederruflich für Sie verloren und muss neu erstellt werden.

Mögliches verfahren um als Admin die Daten anderer wieder zu erlangen:

  • CSV des Nutzers und seiner Übungen exportieren
  • Nutzer löschen
  • Nutzer mit denselben Daten wieder erstellen
  • CSV wieder importieren

    • Hier wird wieder ein neues Passwort festgelegt und kann weitergegeben werden

    Sollte der Admin sein Passwort vergessen, muss die Datenbank neu Aufgesetzt werden, das heißt ein komplett Reset der Webapplikation.

    Passwort ändern

    Wenn Sie sich an Ihr Passwort errinern, können Sie es in der Trainerübersicht unter Ihrem eigenen Nutzer ändern.

    Abmelden

    Um sich Abzumelden, klicken Sie unten rechts in der Navigatorleiste.
    Sie können dann ihren Account wechseln oder sich erneut anmelden.

    Den eigenen Nutzer zu löschen ist nicht möglich, dies kann nur der Admin.
    Der Admin kann sich nicht selbst löschen

    Athleten

    Athleten erstellen

    Um ein neues Konto zu erstellen, müssen Sie entweder als Admin oder als Trainer angemeldet sein und sich auf der Athletenübersicht befinden.
    Hier können Sie jetzt entweder mit in der Seitenleiste links oder mit beziehungsweise neben der Suchleiste einen neuen Athleten erstellen.

    Einen Nutzer können Sie entweder Erstellen indem Sie alle benötigten Daten in die vorhandenen Textfelder einpflegen oder per CSV Upload gleich mehrere Nutzer auf einmal erstellen.

    Bedenken Sie, keine Umlaute und Sonderzeichen wie ä, ö, %... in der E-Mail Adresse und im Nutzernamen zu verwenden, sowie sicherzustellen, dass das eingegebene Datum existiert.

    !Bitte denken Sie daran, das Passwort des neu erstellten Nutzers zu kopieren, da es auf keine andere Weise abgerufen werden kann.
    Nach dem ersten Login können diese, wie in Abschnitt "Login" beschrieben, Ihr Passwort ändern!

    Das Format der CSV ist:

    Vorname Nachname E-Mail Geburtsdatum Geschlecht
    Text Text Text: Format ...@... TT.MM.JJJJ M/W

    Das Trennzeichen der CSV ist das Semicolon ";".

    Athleten welche per CSV erstellt werden bekommen momentan ein zufälliges Passwort zugewiesen welches nicht eingesehen werden kann.
    Da Athleten momentan sowieso nicht die Webapplikation nutzen sollen, ist dies kein Problem.
    Um einfache Zukünftige implementierung von Athleten zu ermöglichen, ist es jedoch möglich ein Passwort für Athleten in dem integrierten Nutzer Erstellungssystem zu setzen.

    Athleten finden

    Um einen Athleten zu finden, können Sie die Suchleiste oben in der Seitenleiste links benutzen.
    Die Suchleiste sucht nach dem Vor- und Nachnamen des Athleten und zeigt Ihnen alle Athleten an, die den eingegebenen Text in entweder Vor- oder Nachnamen enthalten.

    Athleten bearbeiten

    Einzelne Athleten können Sie ebenso in der Athletenübersicht bearbeiten. Hierfür müssen Sie nur einen Athleten in der Leiste links anklicken und dann oben rechts auf .
    Hier können Sie alle persönlichen Daten des Athletens anpassen, außer deren Passwort.

    Auf dem Knopf "Details" direkt neben "Bearbeiten" können Sie einen Athleten auch löschen.

    Daten Erfassen

    Damit Sie einem Athleten seine erreichten Werte eintragen können müssen Sie genau wie bei den anderen Schritten in die Athletenübersicht wechseln.
    Wenn Sie hier einen Athleten auswählen sehen Sie in der Mitte des Bildschirms eine Mitteilung, dass noch keine Einträge vorhanden sind, oder die bereits eingetragenen Übungsdaten des Athleten.
    Hier können Sie auf klicken um ein neues Modal zu öffnen.
    In diesem Modal können Sie sehr Intuitiv auswählen welche Übung aus welcher Kategorie passend zur Altersklasse des Athleten erfassen wollen.
    Haben Sie die Übung gefunden müssen Sie nurnoch die passenden Werte eingeben. Das neue Fenster passt sich automatisch an die bedürfnisse der Übung an.
    z.B. haben Sprints ein Feld um die Zeit zu erfassen und Seilspringen ein Feld um die Anzahl der Wiederholungen einzutragen.

    Sobald Sie das geschafft haben, wird die Übung in der Datenbank eingetragen und in der Liste der Erfassten Übungen angezeigt.

    Auch hier können Daten per CSV eingepflegt werden.

    Das Format der CSV ist:

    Name Vorname Geschlecht Geburtsjahr Geburtstag Übung Kategorie Datum Ergebnis Punkte DBS
    Text Text M/W JJJJ TT.MM Text Text TT.MM.JJJJ Wert / Text Zahl Ja/Nein

    Das Trennzeichen der CSV ist das Semicolon ";".

    Beachte hier, dass die Werte stimmen müssen und nicht existente Übungen automatisch erstellt werden.

    Übungen welche am selben Tag wie bereichts vorhandene gleiche Übungen eingetragen werden, überschreiben diese.

    Erfasste Übungen einsehen

    In der Ansicht eines Athletens können Sie nach dem erfassen mindestens einer Übung eine Liste der erfassten Übungen sehen.
    Die erfassten Übungen können Sie jetzt mittels der Filter funktion oben rechts so Filtern, dass nur relevante Übungen angezeigt werden.
    Die Sortieren Funktion finden Sie auf den grauen Überschriften über der Liste, klicken Sie einfach an nach was Sie sortieren wollen. Ein erneuter Klick auf dieselbe Kategorie ändert die Sortierreihenfolge.

    Erfasste Übungen bearbeiten

    Wenn Sie sich in der Übersicht eines Athletens befinden und dessen Übungen sehen, können Sie diese auch bearbeiten.
    Klicken Sie dafür einfach auf die drei Punkte rechts neben der Übung und wählen Sie "Bearbeiten ".
    Hier können Sie die Werte der Übung anpassen und speichern.
    Sollten Sie eine Übung löschen wollen, können Sie dies ebenfalls über die drei Punkte machen.

    Trainer verwalten

    Trainer erstellen

    In der Trainerübersicht können Sie wie in der Athleten Ansicht alle Trainer sehen und nach ihnen suchen. Auch hier können Sie mit neben der Suchleiste oder dem unter den vorhandenen Trainern einen neuen Trainer erstellen.

    Trainer bearbeiten

    Wenn Sie hier auf einen Trainer klicken, können Sie diesen bearbeiten oder löschen.

    Beim bearbeiten können Sie alle Daten des Trainers bis auf deren Passwort ändern.

    Beim Löschen eines Trainers werden alle Daten des Trainers gelöscht und können nicht wiederhergestellt werden.
    Die Erfassten Übungen des Trainers bleiben vorhanden.

    Übungskatalog

    Übungen einsehen

    Im Übungskatalog können Sie alle Übungen sehen, die in der Datenbank vorhanden sind.
    Diese können Sie vor dem ersten Start in der values.json im Backend festlegen.
    Standardmäßig sind dort alle Übungen der Altersklassen 6-18 im Jahr 2024 eingetragen.

    Hier können Sie wie bei der Übersicht eines Athleten nach den Überschriften, durch anklicken dieser, sortieren und einen Filter anwenden um nur relevante Übungen anzeigen zu lassen.

    Die Suchfunktion ist in dieser Übersicht sehr ausgefeilt, denn es kann nach allem gesucht werden.

  • Name
  • Benötigte Werte für eine Medaille
  • Kategorie
  • Geschlecht
  • ...
    • Tippen Sie einfach ein nach was Sie suchen. Zusammen mit dem Filter und der Sortierung finden Sie garantiert alles was Sie brauchen.

    Übungen bearbeiten

    Falls Sie die Übungen aktuell halten wollen, können Sie deren Werte bearbeiten. Klicken Sie dafür einfach die drei Punkte rechts an der Übung.

    Das Löschen einer Übung ist auch möglich, bedenken Sie hier, dass das unwiederruflich ist.

    Übungen hinzufügen

    Um Übungen hinzuzufügen, klicken Sie einfach oben rechts auf "Disziplin hinzufügen", hier können Sie dann alle benötigten Daten einer Übung eintragen.

    Es liegt in Ihrer verantwortung, die Übungen korrekt einzutragen, sodass Sie mit den offiziellen Übungen übereinstimmen.

    Datenschutz

    Diese Webapplikation ist dazu gedacht Lokal ausgeführt zu werden und von nur einer Gruppe von Personen für testzwecke genutzt zu werden.

    Es wird keine Internetverbindung benötigt, da es keine Daten senden / empfangen muss.
    Somit ist es uns erlaubt, dass alle Trainer alle Athleten sehen und bearbeiten können. Auch die Erstellung von Nutzern mittels deren persönlichen Daten ist in diesem Rahmen kein Problem.
